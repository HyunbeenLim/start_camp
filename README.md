## 로컬 -> 원격 저장소

- 경로 설정

```sh
git remote add origin https://github.com/HyunbeenLim/start_camp.git
```

- 커밋 보내기
```sh
git push -u origin 
```

##  원격 저장소 -> 로컬

```sh
git pull remote_repo_url
```

```sh
git clone remote_repo_url
# 강의장 -> 깃헙 -> 집에서 유용함. 모든 기록, 파일 가져와줌
```
